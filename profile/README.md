# Mapy.cz was renamed to Mapy.com

[Mapy.com](https://mapy.com/) is a Czech web and mobile mapping application developed by Seznam.cz, offering worldwide map coverage, route planning, and additional features for tourists and drivers.
Developers can integrate Mapy.cz data and functionalities into their applications, refer to [mapycom/developer](https://github.com/mapycom/developer) or [developer.mapy.com](https://developer.mapy.com/) for more information.
